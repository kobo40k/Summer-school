# SOC Analyst — Летняя Школа 2025

Интерактивный лендинг, посвящённый профессии **Специалист SOC (первая линия)**.

## О проекте

Сайт объясняет профессию SOC-аналитика так, чтобы было понятно:
- Чем человек занимается
- Зачем он нужен
- Как туда попасть

## Структура сайта

- **Главный экран** — название профессии и ключевые метрики
- **О профессии** — кто такой SOC-аналитик простыми словами
- **Рабочие задачи** — 7 основных задач с интерактивными карточками
- **Hard & Soft Skills** — навыки, разделённые на технические и软ые
- **День из жизни** — таймлайн рабочего дня
- **Карьера** — пошаговый план для студентов
- **Тест** — интерактивная викторина из 4 вопросов

## Технологии

- React.js (Vite)
- CSS3 (анимации, градиенты, CSS Grid)
- Мобильная адаптивность

## Запуск

```bash
cd pentester-react
npm install
npm run dev
```

## Деплой

Сборка: `npm run build`
Статические файлы в `dist/`
